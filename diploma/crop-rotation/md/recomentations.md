# Общий принцип редактуры

Сейчас у тебя уже есть хороший каркас. Основная задача дальше — не дописывать “ещё больше текста”, а сделать текст **плотнее и чище**.

Главные повторы, которые у тебя почти наверняка гуляют по нескольким главам:

* работа — **не оптимизатор севооборота**, а explainable recommendation / decision-support prototype;
* **baseline CatBoost** — финальная модель;
* **Top-k важнее top-1**;
* **KG не заменяет модель**, а интерпретирует shortlist;
* spatial demo — **демонстрационный слой**, а не новая модель.

Эти тезисы нужны, но каждый из них лучше **полноценно объяснить один раз**, а дальше только коротко напоминать.
